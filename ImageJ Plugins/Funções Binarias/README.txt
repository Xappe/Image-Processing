Plugin with morphological operations on binary images. Implement dilation, erosion, closing, opening and
border (outline) for binary images. (these functions already exist inside ImageJ)

The structuring element used was a square with dimensions 3 x 3 for all techniques.



Plugin com operações morfológicas em imagens binárias. Implementar as operações de dilatação, erosão, fechamento, abertura e 
borda (outline) para imagens binárias. (essas funções ja existem dentro do ImageJ)

O elemento estruturante utilizado foi um quadrado com dimensões 3 x 3 para todas as técnicas.
