The Plugin "Converter_RGB_Escala_Cinza_" transforms an RGB image into three grayscale images. Each image represents an RGB channel, so
three images with different shades of gray will be generated.

The Plugin "Converter_RGB_Escala_Cinza_Media_Aritmetica_" transforms an RGB image into a single image in grayscale, being the tone of the image
calculated through the arithmetic average of the values ​​in the RGB channels in that pixel.

Plugin "IG_Converter_RGB_Scala_Cinza_" contains three functions that can be toggled through a graphical interface. they being the function of average
arithmetic already presented, analog Luminance and digital Luminance

The Plugin "Converter_RGB_Escala_Cinza_Luts_" transforms three 8bit-s images into a colored image with .gif extension. this extension
uses only one color channel but uses a matrix of colors detected in the image (LookUp Table)



O Plugin "Converter_RGB_Escala_Cinza_" transforma uma imagem RGB em três imagens em escala de cinza. Cada imagem represeta um canal RGB, logo 
será gerado trés imagens com diferentes tons de cinza.

O Plugin "Converter_RGB_Escala_Cinza_Media_Aritmetica_" transforma uma imagem RGB em uma unica imagenm em escala de cinza, sendo o tom da imagem 
calculada atraves da media aritimetica dos valores nos canais RGB naquele pixel.

O Plugin "IG_Converter_RGB_Escala_Cinza_" contem tres funções que podem ser alternadas por uma interface grafica. sendo elas a funcao de media
aritmetica ja apresentada, Luminance analogico e Luminance digital

O Plugin "Converter_RGB_Escala_Cinza_Luts_" transforma tres imagens 8bit-s em uma imagem colorida de extensao .gif. Essa extensão
utiliza apenas um canal de cor porem utiliza uma matriz de cores detectadas na imagem (LookUp Table)
