Plugin "Rotular_Componentes_Conexos_" identifies connected areas and colors them with alternating tones.
These fields can be better seen through an ImageJ tool called "ROIs"


Plugin "Rotular_Componentes_Conexos_" identifica areas conexas e as pinta de cores com tons alternados.
Esses campos podem ser melhor vistos atraves de uma ferramenta do ImageJ chamada de "ROIs"
