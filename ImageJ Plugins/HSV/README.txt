Plugin performs image segmentation using the properties of the HSV color pattern.
6 sliders, 2 for each of the properties (Hue, Saturation and Brightness).
It is recommended to test the image of the HSV color cylinder, as it is possible to see the change of each property more clearly.


Plugin realiza a segmentação de imagens utilizando as propriedades do padrão de cores HSV.
6 sliders, 2 para cada uma das propriedades (Matiz, Saturação e Brilho).
Recomenda-se para teste a imagem do cilindro de cores HSV, pois é possivel ver a mudança de cada propiedade com mais clareza.
