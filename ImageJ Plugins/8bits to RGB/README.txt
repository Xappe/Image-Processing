The plugin will generate an RGB image by merging the 8-bit images

Note: If the 8-bit images are not inserted in the "RGB" order, the values ​​of each color channel will be changed. That is, for the image to come out with the correct result, it is necessary that the images are inserted in the order where the first image represents the RED scale, the second the GREEN scale and the last one in the BLUE scale.


O plugin irá gerar uma imagem RGB através da junção das imagens 8-bits

Obs: Caso as imagens 8-bits não sejam inseridas na ordem "RGB" havera a troca dos valores de cada canal de cor. Ou seja, para que a imagem sair com o resultado correto, é nescessario que as imagens sejam inseridas na ordem onde a primeira imagem representa a escala RED, a segunda a esclaa GREEN e a ultima na escala BLUE.
