Plugin changes the brightness and contrast values ​​of an image, as well as applying solarization and desaturation techniques to it.

When the "ok" button is pressed, the image characteristics will be changed definitively, if the "cancel" button is pressed
the image will return to its original characteristics.


Plugin altera os valores de brilho e contraste de uma imagem, bem como aplicar as técnicas de solarização e dessaturação na  mesma.

Quando pressionado o botão "ok" as características da imagem serão alteradas de forma definitiva, caso seja pressionado o botão "cancel" 
a imagem voltará para as suas caraterísticas originais.
